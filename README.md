## 秋秋次元-原生js动漫搜索网站
在线预览网址：https://qiuqiu-dimension.qiuyun.top/

## 主要功能
- 主题切换（亮色/暗色模式）
- 每日放送（聚合每日动漫并渲染卡片）
- 智能搜索（输入关键词搜索动漫，500ms防抖）
- 动漫详情（点击卡片弹出模态框，显示原名、开播日期、简介）
- 评分排序（支持按评分从高到低排序推荐列表）
- 收藏功能（添加/移除收藏，收藏列表独立显示）
- 轮播图（自动/手动切换，小圆点指示，鼠标悬停暂停）

- ## 技术栈
- HTML5 + CSS3（Flex/Grid布局、CSS变量、过渡动画）
- 原生 JavaScript
  - fetch + async/await 异步请求
  - 防抖函数、事件委托、类名切换
  - localStorage 数据持久化
  - 模板字符串、数组方法（map、filter、forEach、some）
